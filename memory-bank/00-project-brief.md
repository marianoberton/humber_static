# Project Brief: Humber Static

## Objetivo
Sitio web estático para Humber Internacional, enfocado en logística y transporte internacional.

## Alcance
- Landing page en Español (`index.html`) y Portugués (`index-pt.html`).
- Formulario de contacto integrado.
- Diseño responsivo con Tailwind CSS.

## Stack
- HTML5
- Tailwind CSS (CDN)
- Vanilla JS
