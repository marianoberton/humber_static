# Architecture

## Estructura
- Raíz: Archivos HTML (`index.html`, `index-pt.html`).
- `assets/`: Imágenes, logos, iconos.
- `js/`: Lógica del cliente (`app.js`).
- `css/`: Estilos personalizados (si los hubiera fuera de Tailwind).

## Patrones
- **Single Page Application (SPA) feel**: Navegación por anclas (`#section`).
- **Tailwind CSS**: Estilizado utilitario.
- **Formulario**: Validación en `app.js` y envío (simulado o integrado).
